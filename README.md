# HerringData

Load Pacific Herring (*Clupea pallasii*) data from databases and
wrangle for data summary reports and input to statistical catch-age models.
Data includes biological (number- and weight-at-age), catch, and spawn
index data.

## Data summary reports

Tables, figures, and other content
for annual [data summary reports](https://github.com/grinnellm/Reports).
See the [vignette](xxx) for an introduction to the data summaries.

## Input files for stock assessment

Spatially integrated statistical catch-at-age herring
([SISCAH](https://www.publications.gc.ca/site/eng/9.959363/publication.html))
input files:
biological (number- and weight-at-age),
catch, and
spawn index data.
See the [vignette](xxx) for an introduction to the SISCAH input data.
